Steam-Recommendations-API

Esta página contiene la descripción de los endpoints disponibles para el uso en Steam-Recommendations-API.
Para acceder a cada endpoint se debe enviar una petición GET al servidor, con el formato indicado en cada uno y la respuesta será en formato JSON.

Link a documentación automática de FastAPI: Click aquí

User Data

Endpoint para obtener datos de usuario.

User Data: Recibe un string con el "User Id" y devuelve un summary del user

Para acceder a este endpoint, utiliza la siguiente URL:

https://steam-api-project.onrender.com/userdata/{user_id}

https://steam-api-project.onrender.com/userdata/--000-- Resultado: { "user_id": "--000--", "porcentaje_recomendacion": 100, "cantidad_gastada": 402.77, "cantidad_items": 58 }

Count Reviews

Endpoint para la cantidad de reviews realizadas entre dos fechas

https://steam-api-project.onrender.com/countreviews/{date1}/{date2}

https://steam-api-project.onrender.com/countreviews/2011-11-5/2014-07-8 { "results": [ { "cantidad_usuarios": 17072, "porcentaje_recomendacion": 0.91 }] }

Sentiment Analysis

Endpoint para realizar análisis de sentimientos de las reviews de los usuarios agrupadas por año de lanzamiento del video juego.

Para acceder a este endpoint, utiliza la siguiente URL:

https://steam-api-project.onrender.com/sentimentanalysis/{year}

https://steam-api-project.onrender.com/sentimentanalysis/2010 Resultado: { "results": [ { "year_released": "2010", "Negative": 265, "Neutral": 403, "Positive": 1341 }] }

Genre

Endpoint para obtener información sobre un género específico.

Para acceder a este endpoint, utiliza la siguiente URL:

   https://steam-api-project.onrender.com/genre/{genre}

https://steam-api-project.onrender.com/genre/RPG Resultado: { "results": [ { "Genre": "rpg", "Total_Hours": 1041022718, "Rank": 3 }] }

User for Genre

Endpoint para obtener una lista de los usuarios con más horas de juego en un género.

Para acceder a este endpoint, utiliza la siguiente URL:

https://steam-api-project.onrender.com/userforgenre/{genre}

https://steam-api-project.onrender.com/userforgenre/Indie { "results": [ { "user_id": "wolop", "user_url": "http://steamcommunity.com/id/wolop" }, { "user_id": "76561198039832932", "user_url": "http://steamcommunity.com/profiles/76561198039832932" }, { "user_id": "tsunamitad", "user_url": "http://steamcommunity.com/id/tsunamitad" }, { "user_id": "jimmynoe", "user_url": "http://steamcommunity.com/id/jimmynoe" }, { "user_id": "lildoughnut", "user_url": "http://steamcommunity.com/id/lildoughnut" } ] }

Developer

Endpoint para obtener información sobre un desarrollador específico.

Para acceder a este endpoint, utiliza la siguiente URL:

https://steam-api-project.onrender.com/developer/{developer}

https://steam-api-project.onrender.com/developer/Mortis Games { "results": [ { "release_year": 2017, "item_count": 3, "porcentaje_free": 33.33 } ] }

Game Recommendations

Endpoint para obtener recomendaciones de juegos basado en un juego específico.

Para acceder a este endpoint, utiliza la siguiente URL:

https://steam-api-project.onrender.com/game_recommendations/{game_id}

https://steam-api-project.onrender.com/game_recommendations/47810 { "results": [ { "game_id": 47730, "title": "Dragon Age™: Origins Awakening" }, { "game_id": 17450, "title": "Dragon Age: Origins" }, { "game_id": 17460, "title": "Mass Effect" }, { "game_id": 7110, "title": "Jade Empire™: Special Edition" }, { "game_id": 24980, "title": "Mass Effect 2" } ] }
